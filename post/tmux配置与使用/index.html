一个会话(Session)可包含多个窗口(Window),一个窗口(Window)可包含多个窗格(Pane)

tmux中的快捷键前缀

tmux中的快捷键需要先按Ctrl+b前缀然后按相应的键，以下文档中部分省略了Ctrl+b

tmux ls

查询所有会话(Session)

tmux new -s session-name

创建会话

tmux detach | Ctrl+b d

断开当前会话 （断开会话并不影响会话中运行的程序，断开后还可以重新连接）

tmux a -t session-name

重新连接会话，tmux a命令可以快速链接最近断开的会话，tmux a是tmux attach的缩写

tmux kill-session -t session-name

关闭会话(关闭会话后会话中的程序也会全部关闭,关闭会话，会话中的所有窗口，窗格都将会关闭)，可以通过快捷键Ctrl+d关闭当前的会话，窗口或窗格或显式输入exit

tmux kill-window -t window-number

关闭一个window窗口，这在窗口卡死的时候很好使

tmux switch -t session-name

切换会话

Ctrl+b ?

获取帮助文档

Ctrl+b &

关闭当前窗口

会话(Session)管理

  • s 列出所有会话
  • $ 重命名当前会话 | tmux rename-session -t old-session-name new-session-name
  • d 断开当前会话

窗口(Window)管理

  • c 创建一个新窗口
  • , 重命名当前窗口 | tmux rename-window new-window-name
  • w 列出当前会话所有窗口
  • % 将当前窗口分成左右两个 | tmux split-window -h
  • " 将当前窗口分成上下两个 | tmux split-window
  • 0~9 选择0-9对应的窗口
  • n 选择下一个窗口
  • p 选择上一个窗口

窗格(Pane)管理

  • q 显示窗格编号
  • o 在窗格间切换
  • ! 在新窗口中显示当前窗格
  • x 关闭当前窗格
  • } 与下一个窗格交换位置
  • { 与上一个窗格交换位置

# 快速实现窗格切换

alias R="tmux select-pane -R"
alias L="tmux select-pane -L"

# 快速交换窗格

alias DD="tmux swap-pane -D"

# 通过别名快速切换到test会话
alias tt="tmux switch -t test"

使用遇到的问题

设置set-option -g prefix2 ` , 导致反引号不能使用了

解绑： 在~/.tmux.conf中添加如下

1
2

set -gu prefix2
unbind `

刷新tmux配置

  1. 配置完~/.tmux.conf以后，重启tmux起效
  2. 先按C+b，然后输入：，进入命令行模式， 在命令行模式下输入： source-file ~/.tmux.conf
  3. 在命令行输入tmux source-file ~/.tmux.conf
  4. 在配置文件中加入下面这句话，以后改了只需要按前缀+r了。 将r 设置为加载配置文件，并显示"reloaded!“信息 bind r source-file ~/.tmux.conf \; display "Reloaded!"
